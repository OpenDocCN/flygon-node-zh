## 下载

### Docker

```
docker pull apachecn0/freelearn-node-zh
docker run -tid -p <port>:80 apachecn0/freelearn-node-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-node-zh
freelearn-node-zh <port>
# 访问 http://localhost:{port} 查看文档
```
